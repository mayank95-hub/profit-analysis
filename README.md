# 📊 Profit Analysis Dashboard

## 📌 Project Overview
This project analyzes startup profit using Power BI. The dashboard helps identify how R&D Spend, Marketing Spend, and Administration Cost affect Profit.

---

## 🛠 Tools Used
- Power BI
- SQL
- Excel
- DAX

---

## 📂 Project Files
- Profit Analysis Dashboard (.pbix)
- SQL Script (.sql)
- Presentation (.pptx)

---

## 📈 Dashboard Features
- KPI Cards
- Profit by State
- R&D Spend Analysis
- Marketing Spend Analysis
- Interactive Filters

---

## 📊 Key Insights
- Higher R&D investment generally increases profit.
- Marketing has a positive impact on profit.
- State-wise profit comparison is available.
